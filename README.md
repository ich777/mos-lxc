# MOS LXC

MOS LXC provides **built and packaged LXC binaries** for use within the MOS ecosystem.

This repository contains the **build scripts, packaging logic, and configuration files**
required to compile LXC from upstream sources and package it for MOS.

---

## Overview

LXC is built **from upstream source code** and packaged for integration into MOS.

No functional changes to LXC itself are intended.  
The goal of this repository is to provide **consistent, reproducible LXC packages** for MOS installations.
